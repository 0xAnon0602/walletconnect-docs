## defaultChain

You can set a desired chain for the initial connection:

<PlatformTabs groupId="w3m" activeOptions={["html","vue","react"]}>
<PlatformTabItem value="react">

```ts
import { mainnet } from 'wagmi/chains'

createWeb3Modal({
  //...
  defaultChain: mainnet
})
```

</PlatformTabItem>
<PlatformTabItem value="vue">

```ts
import { mainnet } from 'wagmi/chains'

createWeb3Modal({
  //...
  defaultChain: mainnet
})
```

</PlatformTabItem>
<PlatformTabItem value="html">

```ts
import { mainnet } from 'wagmi/chains'

const modal = new Web3Modal({
  //...
  defaultChain: mainnet
})
```

</PlatformTabItem>
</PlatformTabs>



## includeWalletIds

Allows to override default recommended wallets that are fetched from [WalletConnect explorer](https://walletconnect.com/explorer?type=wallet). 
You can define an array of wallet ids you'd like to prioritize (order is respected). 
You can get these ids from the explorer link mentioned before by clicking on a copy icon of desired wallet card.

<PlatformTabs groupId="w3m" activeOptions={["html","vue","react"]}>
<PlatformTabItem value="react">

```ts
createWeb3Modal({
  //...
  includeWalletIds: [
  '1ae92b26df02f0abca6304df07debccd18262fdf5fe82daa81593582dac9a369',
  '4622a2b2d6af1c9844944291e5e7351a6aa24cd7b23099efac1b2fd875da31a0'
  ]
})
```

</PlatformTabItem>
<PlatformTabItem value="vue">

```ts

  //...
  includeWalletIds: [
  '1ae92b26df02f0abca6304df07debccd18262fdf5fe82daa81593582dac9a369',
  '4622a2b2d6af1c9844944291e5e7351a6aa24cd7b23099efac1b2fd875da31a0'
  ]
})
```

</PlatformTabItem>
<PlatformTabItem value="html">

```ts

  '1ae92b26df02f0abca6304df07debccd18262fdf5fe82daa81593582dac9a369',
```

## excludeWalletIds

Allows to exclude wallets that are fetched from [WalletConnect explorer](https://walletconnect.com/explorer?type=wallet).
You can define an array of wallet ids you'd like to exclude. 
You can get these ids from the explorer link mentioned before by clicking on a copy icon of desired wallet card.


<PlatformTabs groupId="w3m" activeOptions={["html","vue","react"]}>
<PlatformTabItem value="react">

```ts
createWeb3Modal({
  //...

  '4622a2b2d6af1c9844944291e5e7351a6aa24cd7b23099efac1b2fd875da31a0'
  ]
```
