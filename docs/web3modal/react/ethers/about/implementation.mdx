You can start Web3Modal configuration using either **default** or **custom** mode.

Default mode will implement WalletConnect, Browser Wallets (injected), [EIP-6963](https://eips.ethereum.org/EIPS/eip-6963) and Coinbase connectors.

<Tabs>
<TabItem value="default" label="Default">

In your `_app.tsx` file set up the following configuration, making sure that all functions are called **outside** the App component.

```tsx
import { createWeb3Modal, defaultConfig } from '@web3modal/ethers5/react'

// 1. Get projectId
const projectId = 'YOUR_PROJECT_ID'

// 2. Set chains
const chains = [1, 42161]

// 3. Create modal
const metadata = {
  name: 'My Website',
  description: 'My Website description',
  url: 'https://mywebsite.com',
  icons: ['https://avatars.mywebsite.com/']
}

createWeb3Modal({
  ethersConfig: defaultConfig({ metadata }),
  chains,
  projectId
})

export default function App() {
  return <YourApp/>
}
```

<TabItem value="custom" label="Custom">


In your `_app.tsx` file set the following configuration, making sure that all functions are called outside the App components.

```tsx
import { createWeb3Modal, defaultConfig } from '@web3modal/ethers5/react'

// 1. Get projectId
const projectId = 'YOUR_PROJECT_ID'

// 2. Set chains
const chains = [1, 42161]

// 3. Create modal
const metadata = {
  name: 'My Website',
  description: 'My Website description',
  url: 'https://mywebsite.com',
  icons: ['https://avatars.mywebsite.com/']
}

createWeb3Modal({
  ethersConfig: defaultConfig({ 
    metadata,
    defaultChainId: 1,
    enableEIP6963: true,
    enableInjected: true,
    enableCoinbase: true,
    rpcUrl: '...' // used for the Coinbase SDK
  }),
  chains,
  projectId
})

export default function App() {
  return <YourApp/>
}
```
