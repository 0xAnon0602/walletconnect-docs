---
title: Onramp
---

Web3Modal SDK integrates Coinbase Pay, which lets users sign-in to their Coinbase account and use their saved payment methods or crypto balance on self-custody wallets or dapps.

:::caution
Onramp is currently in alpha and it's not recommended for production apps.
:::

## Integration

In order to enable Onramp on Web3Modal add the `enableOnramp` flag set to `true` in the `createWeb3Modal` function configuration.

````ts
const modal = createWeb3Modal({
  //...
  chains: [mainnet],
  projectId,
  /* highlight-add-start */
  enableOnramp: true // Optional - false as default
  /* highlight-add-end */
})
```
