# Actions

:::caution
The Web3Modal SDK for Flutter is currently in Beta.
:::

## Launch the current wallet

```dart
await _w3mService.launchCurrentWallet();
```

## Send a request

```dart
await _w3mService.web3App.request(
  topic: _w3mService.session.topic,
  chainId: 'eip155:1',
  request: SessionRequestParams(
    method: 'personalSign',
    params: ['Sign this', '0xdeadbeef'],
  ),
);
```

For more examples check out the [example app](https://github.com/WalletConnect/Web3ModalFlutter/tree/master/example)
