You can start Web3Modal configuration using either **default** or **custom** mode.

Default mode will implement WalletConnect, Browser Wallets (injected), [EIP-6963](https://eips.ethereum.org/EIPS/eip-6963) and Coinbase connectors.

<Tabs>
<TabItem value="default" label="Default">


In your `App.vue` file set up the following configuration.

```html
<script setup>
import { createWeb3Modal, defaultConfig } from '@web3modal/ethers-5/vue'

// 1. Get projectId
const projectId = 'YOUR_PROJECT_ID'

// 2. Set chains
const chains = [1, 42161]

// 3. Create modal
createWeb3Modal({
  ethersConfig: defaultConfig(),
  chains,
  projectId
})
</script>

<template>
  // Rest of your app ...
</template>
```
