### useWeb3ModalAccount

Hook that returns the client's information.

```tsx
import { useWeb3ModalAccount } from '@web3modal/ethers5/vue'

const { address, chainId, isConnected } = useWeb3ModalAccount()
```

### useWeb3ModalSigner

Hook that returns the `signer`, the `walletProvider` and the `WalletProviderType`.

```tsx
import { useWeb3ModalSigner } from '@web3modal/ethers5/vue'

const { signer } = useWeb3ModalSigner()

async function onSignMessage() {
  const signature = await signer?.signMessage('Hello Web3Modal Ethers')
  console.log(signature)
}
```

The **useWeb3ModalSigner** hook returns the following states:
