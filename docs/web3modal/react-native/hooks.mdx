# Hooks

## useWeb3Modal

Control the modal with the `useWeb3Modal` hook

```ts
import { useWeb3Modal } from '@web3modal/viem-react-native'

export default function Component() {

const { open, close } = useWeb3Modal()

open()
```

You can also select the modal's view when calling the `open` function

```ts
open({ view: 'Account' })
```

## useWeb3ModalState

Get the current value of the modal's state

```ts
import { useWeb3ModalState } from '@web3modal/viem-react-native'
```

```ts
const { open, selectedNetworkId } = useWeb3ModalState()
```

## useAccount

Hook for accessing account data and connection status.

```tsx
import { useAccount } from '@web3modal/viem-react-native'

function App() {
  const { address, isConnected } = useAccount()

  if (!isConnected) return <Text>Not connected</Text>
  return <Text>{address}</Text>
}
```
