# Next.js

Web3Modal SDK has support for [Wagmi](https://wagmi.sh/) and [Ethers v5](https://docs.ethers.org/v5/getting-started/).
Choose one of these ethereum libraries to get started.

:::info Note
These steps are using the [Next.js](https://nextjs.org/) app router.
:::

## Installation

<PlatformTabs groupId="eth-lib" activeOptions={["wagmi","ethers"]}>

## Extra configuration

Add the following code in the `next.config.js` file

```ts
// Path: next.config.js
const nextConfig = {
  webpack: (config) => {
    config.externals.push(
      "pino-pretty",
      "lokijs",
      "encoding"
    );
  return config;
}
```
