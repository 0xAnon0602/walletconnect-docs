# Introduction

The Web3Inbox Widget utilizes `iframes` to display Web3Inbox in a widget friendly view, currently making use of WalletConnect's [notify](../../specs/clients/notify/client-api.md) to subscribe and display notifications.

:::caution

Web3Inbox is under development. Want early access? Join our [Pilot Program](https://walletconnect.com/partners)

:::

## Features

Some of the key features of the Web3Inbox Widget include:

- **Inject the account connected to your dapp**
- **Theme how it looks with vanilla css**
- **Subscribe to your dapp and display notifications**
