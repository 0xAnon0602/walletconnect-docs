# Introduction
The Web3Inbox SDK allows one to easily make use of WalletConnect's [notify](../specs/clients/notify/client-api.md) and [chat](../specs/clients/chat/client-api.md) SDKs. This is done by rendering a WebView of the Web3Inbox dapp.

:::caution

Web3Inbox is under development. Want early access? Join our [Pilot Program](https://walletconnect.com/partners)

:::

## Features

Some of the key features of the Web3Inbox SDK include:

- **Manage what SDKs to use and UI pages to show**
- **Display push notification history**
- **Chat with peers**

### Getting Started

To get started with the Web3Inbox SDK, choose the platform you'll be using to build your project. The SDK is currently available for iOS and Android.

<TabItem value="spm">

You can add a Web3Inbox SDK to your project with Swift Package Manager. In order to do that:

1. Open XCode
2. Go to File -> Add Packages
3. Paste the repo GitHub URL: https://github.com/WalletConnect/WalletConnectSwiftV2
4. Tap Add Package
5. Select Web3Inbox check mark

</TabItem>
<TabItem value="cocoa">

1. Update Cocoapods spec repos. Type in terminal `pod repo update`
2. Initialize Podfile if needed with `pod init`
3. Add pod to your Podfile:

```Ruby
pod 'WalletConnectSwiftV2/Web3Inbox'
```

4. Install pods with `pod install`

If you encounter any problems during package installation, you can specify the exact path to the repository

```Ruby
pod 'WalletConnectSwiftV2/Web3Inbox', :git => 'https://github.com/WalletConnect/WalletConnectSwiftV2.git', :tag => '1.5.0'
```
