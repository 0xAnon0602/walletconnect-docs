# Introduction

The Web3Inbox SDK allows one to easily make use of WalletConnect's [Notify](../api/notify/about) and [Chat](../api/chat/about) SDKs to create a seamless messaging experience.

## Features

Some of the key features of the Web3Inbox SDK include:

- **Manage what SDKs to use and UI pages to show**
- **Display push notification history**
- **Chat with peers**
