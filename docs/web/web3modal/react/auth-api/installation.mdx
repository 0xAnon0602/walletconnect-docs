## Implementation

Start by adding `Web3ModalAuth` component at the root of your app.

```tsx
import { Web3ModalAuth } from '@web3modal/auth-react'

function App() {
  return (
    <>
      <HomePage />

      <Web3ModalAuth
        projectId="YOUR_PROJECT_ID"
        metadata={{
          name: 'My Dapp',
          description: 'My Dapp description',
          url: 'https://my-dapp.com',
          icons: ['https://my-dapp.com/logo.png']
        }}
      />
    </>
  )
}
```

## Use sign in hook

Set up and use signIn method to open the modal and start user authentication.

```tsx
import { useSignIn } from '@web3modal/auth-react'

function HomePage() {
  const { signIn, data, error, loading } = useSignIn({ statement: 'Connect to my dapp' })

  async function onSignIn() {
    const data = await signIn()
    console.info(data)
  }

  return <button onClick={onSignIn}>Sign in with wallet</button>
}
```
