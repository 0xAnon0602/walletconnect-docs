# Hooks

## useSignIn

Hook to create sign in method with given options. Returns authentication data.

### Example

```tsx
import { useSignIn } from '@web3modal/auth-react'

const { signIn, data, error, loading } = useSignIn(SignInOptions)
```

### Reference

```ts
signIn: (options?: SignInOptions) => Promise<SignInData>
```

```ts
data: SignInData
```

```ts
error: Error | unknown
```

```ts

```
