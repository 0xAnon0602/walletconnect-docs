# Theming

## themeMode

By default [`themeMode`](./options#thememode-optional) option will be set to user system settings i.e. 'light' or 'dark',
however you can easily override it to match design of your dapp.

### Example

```tsx
await EthereumProvider.init({
  qrModalOptions: {
    themeMode: 'dark'
  }
})
```

## themeVariables

Web3Modal's theming is done via css variables. You can override any of them to match your dapp's design via [`themeVariables`](./options#themevariables-optional) option.

### Example

```tsx
EthereumProvider.init({
  qrModalOptions: {
    themeVariables: {
      '--w3m-font-family': 'Roboto, sans-serif',
      '--w3m-accent-color': '#F5841F'
    }
  }
})
```
